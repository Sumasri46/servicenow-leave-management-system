# Leave Management System - ServiceNow

## Project Overview

A Leave Management System developed using ServiceNow App Engine Studio.

## Features

* Employee Leave Request Creation
* Leave Approval Workflow
* Status Tracking
* Manager Comments
* Automated Flow Execution

## Technologies Used

* ServiceNow App Engine Studio
* Flow Designer
* Custom Tables
* Forms

## Workflow

Employee submits leave request
↓
Status automatically changes to Pending
↓
Approval process starts
↓
Status changes to Approved

## Project Components

* Employee Leave Request Table
* Custom Fields
* Forms
* Approval Flow

## Test Results

Flow execution completed successfully with:

* Update Record
* Ask For Approval
* Status Update
